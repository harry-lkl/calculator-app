# calculator-app
a simple in-browser calculator app with a neo brutalism inspired UI, written in JS

basic functions:
    detect click and keypress
    clear all entry (C)
    clear entry (CE)
    backspace
    calculation:
        add
        subtract
        multiply
        divide
        enter
        1/x
        x^2
        sqrt(x)
        decimal
        +/-
    memory (maybe):
        memory clear (MC)
        memory recall (MR)
        memory+ (M+)
        memory- (M-)
        memory store (MS)
